## 剪切板操作

****

对于文本：

- 复制：`command + c`
- 粘贴：`command + v`
- 剪切：`command + x`

对于文件：

- 复制一个副本：`command + d`
- 复制：`command + c`
- 粘贴：`command + v`
- 剪切：先 `command + c` 要剪切的文件，再 `command + option + v` 对应位置

****

## 屏幕操作

****

- 窗口最小化：`command + m`
- 快速切换应用程序：`command + tab`
- 快速关闭当前窗口：`command + q`
- 强制关闭应用程序：`command + option + esc`

****
